# 🚀 GitHub Pages Setup Guide

## Quick Setup Steps:

1. **Go to Repository Settings**
   - Navigate to: https://github.com/hudsonargollo/caixapreta-stack/settings/pages

2. **Configure GitHub Pages**
   - Source: Select "GitHub Actions"
   - This will enable the workflow we just created

3. **Wait for Deployment**
   - The workflow should automatically trigger
   - Check the Actions tab for deployment progress

## Expected URLs:

Once deployed, your Matrix installation guide will be available at:

- **Main URL**: `https://hudsonargollo.github.io/caixapreta-stack/`
- **Matrix Guide**: `https://hudsonargollo.github.io/caixapreta-stack/matrix.html`
- **Simple Guide**: `https://hudsonargollo.github.io/caixapreta-stack/simple.html`
