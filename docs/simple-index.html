Guia Completo de Instalação e Configuração

Instalação Rápida (5 Minutos)

1

Conecte no seu servidor

Acesse seu VPS via SSH como usuário root

ssh root@SEU_IP_DO_SERVIDOR
2

Baixe o script de instalação

Execute os comandos abaixo para baixar e preparar o script

wget https://raw.githubusercontent.com/hudsonargollo/caixapreta-stack/main/caixapreta-stack.sh
chmod +x caixapreta-stack.sh
3

Execute a instalação

Inicie o processo automatizado

sudo ./caixapreta-stack.sh
4

Configure os dados solicitados

O script pedirá duas informações importantes:

  • • Seu domínio principal (ex: meusite.com)
  • • Seu e-mail para certificados SSL
5

Configure DNS

Aponte os subdomínios para o IP do seu servidor:

  • • n8n.seudominio.com
  • • mega.seudominio.com
  • • evolution.seudominio.com
  • • portainer.seudominio.com
  • • traefik.seudominio.com
  • • minio.seudominio.com
  • • grafana.seudominio.com
