5-Minute Installation Instalação em 5 Minutos

Follow these steps in order. The automated script handles the heavy lifting. Siga estes passos em ordem. O script automatizado faz o trabalho pesado.

01
Connect to Your Server Conecte ao Seu Servidor

Access your VPS via SSH as root user. Replace with your server's IP address. Acesse seu VPS via SSH como usuário root. Substitua pelo endereço IP do seu servidor.

TERMINAL
# Connect via SSH ssh root@SEU_IP_DO_SERVIDOR
02
Download the Installation Script Baixe o Script de Instalação

Download and prepare the CaixaPreta Stack installer from the official repository. Baixe e prepare o instalador da CaixaPreta Stack do repositório oficial.

TERMINAL
# Download installer wget https://raw.githubusercontent.com/hudsonargollo/caixapreta-stack/main/caixapreta-stack.sh chmod +x caixapreta-stack.sh
03
Run the Installation Execute a Instalação

Launch the automated installation process. The script will prompt for your domain and email. Inicie o processo de instalação automatizado. O script solicitará seu domínio e e-mail.

TERMINAL
# Run installer as root sudo ./caixapreta-stack.sh
04
Configure DNS Records Configure os Registros DNS
