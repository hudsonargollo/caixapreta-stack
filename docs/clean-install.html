INSTALL GUIDE v2 GUIA DE INSTALAÇÃO v2

Installation Protocol Protocolo de Instalação

Deploy your complete infrastructure stack in 5 minutes. Follow the steps below. Implante sua stack completa de infraestrutura em 5 minutos. Siga os passos abaixo.

🔧 Your Server IP Address 🔧 Endereço IP do Seu Servidor

Enter your VPS IP address to generate personalized installation commands Digite o endereço IP do seu VPS para gerar comandos de instalação personalizados

No IP set yet Nenhum IP definido ainda
1
Connect to Your Server Conecte ao Seu Servidor

Access your VPS via SSH as root user. Acesse seu VPS via SSH como usuário root.

SSH
ssh root@YOUR_IP_ADDRESS
2
Download Installation Script Baixe o Script de Instalação

Download and prepare the CaixaPreta Stack installer from the official repository. Baixe e prepare o instalador da Stack CaixaPreta do repositório oficial.

BASH
curl -fsSL https://raw.githubusercontent.com/hudsonargollo/caixapreta-stack/main/scripts/install/caixapreta-stack-enhanced.sh -o installer.sh && chmod +x installer.sh
3
Run the Installer Execute o Instalador

Execute the installation script. You'll be prompted for your domain and email. Execute o script de instalação. Você será solicitado a fornecer seu domínio e email.

BASH
./installer.sh
4
Configure Your Domain Configure Seu Domínio

Point your domain DNS records to your server IP address. The installer will provide specific instructions. Aponte os registros DNS do seu domínio para o endereço IP do seu servidor. O instalador fornecerá instruções específicas.

DNS A RECORD
*.seudominio.com A YOUR_IP_ADDRESS
5
Wait for SSL Certificates Aguarde os Certificados SSL

SSL certificates are generated automatically via Let's Encrypt. This typically takes 5-15 minutes. Os certificados SSL são gerados automaticamente via Let's Encrypt. Isso normalmente leva 5-15 minutos.

✓ Once complete, your entire stack will be live and accessible! ✓ Após a conclusão, sua stack completa estará ativa e acessível!
