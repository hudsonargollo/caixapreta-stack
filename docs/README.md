# CaixaPreta Stack - Guia de Instalação

Guia interativo e protegido por senha para instalação da infraestrutura CaixaPreta Stack.

Este guia faz parte do repositório principal do CaixaPreta Stack.

## Características

- 🔐 Proteção por senha (`caixapretastack2626`)
- 🎨 Interface moderna com shadcn/ui
- ⚡ Animações suaves com Framer Motion
- 📱 Design responsivo
- 🚀 Deploy automático no Cloudflare Pages

## Tecnologias

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Lucide Icons

## Desenvolvimento

```bash
cd docs
npm install
npm run dev
```

## Deploy

Este projeto está configurado para deploy automático no Cloudflare Pages em `caixapreta.clubemkt.digital/instalar`.

## Estrutura

- `/docs` - Aplicação Next.js do guia de instalação
- `/caixapreta-stack.sh` - Script principal de instalação
